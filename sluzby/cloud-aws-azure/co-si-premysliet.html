Cloud (AWS/Azure/GCP): čo si premyslieť pred návrhom

Cloud nie je len hosting. Je to operačný model. Pred návrhom si ujasni prístupy, zmeny, bezpečnosť a obnovu.

Kontakt

Otázky, ktoré si treba zodpovedať

  • Kto má admin/root a ako sa prístupy auditujú?
  • Ako sa robia zmeny (IaC vs kliky)?
  • Ako vyzerá sieťový model (segmentácia, routing, hranice)?
  • Máme definované RTO/RPO a reálne testy obnovy?
  • Máme monitoring a logy na úrovni služby, nie len VM?

Najväčšie riziká v cloude sú procesné: prístupy, zmeny a chýbajúce rozhodnutia.

Dobrý návrh definuje minimálny štandard a umožní rast bez chaosu.
